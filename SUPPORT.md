# Support

Looking for help with a Yii.Rocks package?

- **Documentation** — start at [yii.rocks](https://www.yii.rocks/) for an overview of all packages.
- **Bugs & feature requests** — open an issue on the relevant package's repository using the provided templates.
- **Usage questions** — open an issue on the relevant repository; there's no separate chat or forum at this time.

Please don't use issues for reporting security vulnerabilities — see [SECURITY.md](SECURITY.md) instead.
